# Lab 7: Installing Storage Explorer

**Objective:** Install Microsoft Azure Storage Explorer and connect to your Azure account.

---

## Steps

1. Sign in to **TestMGMTVM** as **TestAdmin** with password `Github@25`.  

2. Open **File Explorer** → double-click `DVD Drive (D:) NCOTMS01` → open `Tools`.  

3. Double-click `StorageExplorer.exe` to start installation.  

4. In the **Select Setup Install Mode** dialog, click **Install for all users**.  

5. In the **User Account Control** window, click **Yes**.  

6. On the **License Agreement** page, select **I accept the agreement** → click **Install**.  

7. On **Select Destination Location**, click **Next >**.  

8. On **Select Start Menu Folder**, click **Next >**.  

9. On **Completing the Microsoft Azure Storage Explorer Setup Wizard**, click **Finish**.  

10. In the **Microsoft Azure Storage Explorer** dialog, click **OK**.  

11. In the **Connect to Azure Storage** window, click **Next**.  

12. In the **Sign in to your account** window, sign in to Microsoft Azure as `az104username@Gurpreet708.com` with password `Github@25`.  

13. On the **Account Management** page, click **Apply** and review settings.  

14. Close all open windows.
