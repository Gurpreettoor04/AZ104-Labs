# Lab 8: Taking Snapshots

**Objective:** Learn how to take snapshots of virtual machines in VMware Workstation.

---

## Steps

1. Sign in to **TestMGMTVM** as **TestAdmin** with password `Github@25`.  

2. Shut down the virtual machines carefully before taking a snapshot.  

3. In **VMware Workstation**, right-click the virtual machine tab → hover over **Snapshot** → click **Take Snapshot**.  

4. In the **Take Snapshot** dialog box, in the **Name:** text box, type `Lab 8`.  

5. Click **Take Snapshot**. Wait for the process to complete.  

6. Confirm the snapshot is created successfully by checking the snapshot manager (optional).  

---

**Technologies Used:**  
- VMware Workstation  

**Learning:**  
- How to take and manage snapshots of virtual machines for backup and testing purposes.
